# Simulador de Incentivo Fiscal — FUNEPP (Nestlé)

Projeto desenvolvido na **Nestlé** para a **FUNEPP**, entidade de previdência privada da empresa, com o objetivo de incentivar e orientar participantes sobre como contribuições podem impactar o **imposto de renda** e o planejamento financeiro de forma estratégica.

Este simulador foi criado para tornar um tema complexo mais acessível, ajudando usuários a visualizarem, na prática, como aumentar a contribuição para a previdência pode gerar benefícios fiscais e maior organização financeira no longo prazo.

## Contexto e desenvolvimento

Este foi meu primeiro projeto corporativo desenvolvido **do zero** dentro da Nestlé.  
Fui responsável por todo o processo de construção do sistema, desde o estudo inicial até a implementação final.

Para transformar regras reais em uma aplicação funcional, aprofundei meus conhecimentos em temas como:

- Imposto de Renda  
- IRRF e INSS  
- Previdência privada e incentivo fiscal  
- Lógica de cálculo aplicada ao desenvolvimento web  

Além da parte técnica, o desafio também envolveu pensar na experiência do usuário: como apresentar informações tributárias de forma clara, simples e intuitiva para qualquer pessoa.

---

## Tecnologias utilizadas

A implementação foi realizada utilizando exclusivamente:

- HTML  
- CSS  
- JavaScript  

O projeto exigia o uso de **JavaScript como linguagem principal**, por ser a tecnologia homologada e adequada ao ambiente da aplicação, o que me levou a explorar e aplicar ao máximo sua lógica e estrutura.
